# FlowForge Status Dashboard

Run these commands and present a unified summary:

1. **Work Status**: `flowforge work status`
2. **Learning Stats**: `flowforge learn stats`
3. **Guidance Trust**: `flowforge guidance trust`
4. **Available Agents**: `flowforge agent list`

Present the results as a concise dashboard showing:
- Active/pending/completed work items
- Learning progress (short-term and long-term patterns)
- Current trust score and gate stats
- Number of available agents by source
